
# Multi-Agent Epidemic Simulation Tool

This project simulates the impact of different policies on an epidemic using a multi-agent system.

## **Technologies Used**

- **Streamlit**
- **CrewAI**
- **AutoGen**
- **AgentOps**
- **Composio**
- **Together AI**
- **Llama 3.1**
- **Pandas**
- **Plotly**
